Pet Adoption Mobile Application | Animal Reconnect Code

I worked as a sole programmer in a cross-functional team, with a goal of making pet adoption more engaging. We developed a tinder-like mobile application for direct pet adoption; Users can use this to either look for pets or give away their pets. What makes this application different is that it incorporates the swiping mechanism you see in dating apps in order to keep the users' attention. Then, once matched with each other, users can directly communicate with each other via chat.

Used: Flutter, Firebase

PHReddit | Code Privatized

This is the final project for one of my major classes. I worked with a partner to make a replica of Reddit including a functioning backend to persistently store data. Users can register and login with their accounts or as a guest. With an account, users can make a post, reply, or make a community. However, as a guest, they can only read the posts.

Used: MongoDB, Express, React, Node.js

Role-Playing Game | RPG Code

This project was the one that motivated me to pursue Computer Science in college. While I no longer have recordings and screenshots of it, I still remember how proud I was of what I created. To describe what I have made, it is a 2D adventure RPG where the player can interact with various NPCs to buy and sell equipment, complete quests, level up, and encounter random monsters in turn-based combat. Building it taught me how to think critically about implementing key mechanics such as inventory systems, battle logic, managing player and monster stats, etc.

Used: Java, Java Swing
