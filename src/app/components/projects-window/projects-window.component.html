Pet Adoption Mobile Application | Animal Reconnect Code

A mobile application for direct pet adoption between users. Users can use this to either look for pets or give away their pets. What makes this application different is that it incorporates the swiping mechanism you see in dating apps in order to keep the users' attention. Then, once matched with each other, users can directly communicate with each other via chat.

Used: Flutter, Firebase

PHReddit | Code Privatized

Replica of Reddit including a functioning backend. It has a functional database, making it a persistent website. Users can register and login with their accounts or as a guest. With an account, users can make a post, reply, or make a community. As a guest, they can only read the posts.

Used: MongoDB, Express, React, Node.js
